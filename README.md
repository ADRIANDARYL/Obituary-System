# Obituary System Project

## Description

A Flask-based Death Obituary Management System.

## Setup Instructions

1. Clone the repository:

```
git clone https://github.com/ADRIANDARYL/doms-project.git
cd doms-project
```

2. Create virtual environment:

```
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
flask run
```

## Features

* Add obituary records
* View records
* Search records
* Admin panel
